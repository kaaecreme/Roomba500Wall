# Roomba500Wall
Invisible wall made for Roomba500.
Case has been 3D printed.

### Built With
* ATtiny45, programmed using Arduino Mega2560
* 2x 100 resistor
* 1x TSHG6200 Infrared Emitter LED
* 1x Green LED
